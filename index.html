Suggested Resume Bullet

Built an interactive Excel executive dashboard for incident operations, integrating SLA performance, root-cause drivers, and estimated vs actual financial impact with variance trending using PivotTables, PivotCharts, and slicers.

How to use the dashboard

  • Use slicers to filter by System, Priority, Region, Analyst, and Status.
  • Review Estimated vs Actual cost trend to assess forecasting quality.
  • Use variance chart (red/green) to identify months with overruns vs savings.
  • Use root cause chart to prioritize improvements (training, config, bugs, process gaps).
